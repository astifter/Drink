Documentation

Simple Use Case

In the simplest use case the app can log the number of glasses of fluid you drank in a day.

Enabling Reminders

The real power comes from the automatic reminders. To enable them go to the Settings menu and select the Settings / Reminder to enable them. The default settings should be really okay.

Reminders Explained

The reminders start at the time set in First Reminder and then continue each Interval, the number of reminders matches the number of glasses in Target. To space out the reminders evenly the already logged number of glasses is taken into account.

Example: With a First Time of 9:00, Interval of 1:30 and Target of 6 glasses the reminders come at 9:00, 10:30, 12:00, 13:30, 15:00 and 16:30. If you log 4 glasses at 10:00 the next reminder will be at 15:00 (for the 5th glass).

Using the Reminder Window

When the reminder opens its possible to increase the amount of logged glasses and to snooze the reminder. The snooze time depends on the Interval, its 1/3 of this duration. The reminder window that opens after snoozing does not allow you to snooze again, just to be sure.

Example: The 13:30 reminder goes of an is snoozed, the next reminders are scheduled for 14:00 (the snoozed 13:30 one) and for 15:00 (the regular one). When the reminder at 14:00 opens its not possible to snooze again.

Additional Settings

Enabled Auto Dismiss closes the reminder window after 5 minutes.
Enabled Vibrate enables vibration for the reminders.
Manual Correction can be used in case a wrong amout of glasses was logged.
